# Dental B&A 정렬 시스템 규칙

## 1. 정렬 대상
- **안모 (Full Face)**: 눈 간격, 눈 기울기, 눈 중심 기준
- **반모 (Half Face)**: 정중선 길이, 정중선 각도, 입 중심 기준  
- **구강 (Oral)**: 치아 너비, 치아 중심 기준

## 2. 정렬 방식
- **균형 스케일링**: 두 이미지를 중간 크기로 맞춤
  - before: √(ratio) 배율
  - after: 1/√(ratio) 배율
- 위치: 두 이미지 모두 화면 중앙으로 이동
- 각도: 두 이미지 각도 차이를 절반씩 분배

## 3. 출력 형식
- **디졸브 영상**: 1.3초 before → 0.4초 전환 → 1.3초 after (총 3초)
- **좌우 비교 이미지**: BEFORE | 구분선 | AFTER

## 4. 배치 처리
- 여러 쌍의 before/after를 순차 처리
- 개별 결과 파일 생성

## 5. 감지 우선순위
1. MediaPipe 얼굴 감지 시도
2. 얼굴 감지 실패 시 → 구강(치아) 감지
3. 구강 감지 실패 시 → 중앙 정렬 폴백
