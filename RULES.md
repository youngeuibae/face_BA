# Dental B&A 정렬 시스템 규칙

## 1. 정렬 대상
- **안모 (Full Face)**: 눈 간격, 눈 기울기, 눈 중심 기준
- **반모 (Half Face)**: 정중선 길이, 정중선 각도, 입 중심 기준

## 2. 정렬 방식
- 비포 이미지를 애프터 기준으로 정렬 (애프터가 기준 좌표계)
- 스케일: 얼굴 크기 비율 (눈 간격 또는 정중선 길이)
- 회전: 얼굴 기울기 차이 보정 (눈 기울기 또는 정중선 각도)
- 이동: 앵커 포인트를 애프터의 앵커 위치로 이동

## 3. 출력 형식
- **9:16 세로형 PNG**: 얼굴 중심 기준 크롭 (패딩 없음)
- **미리보기**: BEFORE | AFTER 좌우 비교 이미지
- 해상도: 원본 유지 (4096px 상한)

## 4. 처리 파이프라인
1. 비포를 애프터 높이에 맞춰 스케일 (좌표계 통일)
2. MediaPipe FaceMesh로 얼굴 랜드마크 검출
3. 비포를 애프터 기준으로 정렬 (Affine 변환)
4. LAB 색공간에서 밝기 통일
5. 얼굴 중심 기준 9:16 크롭 (동일 영역 → 정렬 유지)
6. 로고 합성 (선택)
7. PNG 저장

## 5. 감지 우선순위
1. MediaPipe 얼굴 감지 시도
2. 감지 실패 시 → 이미지 중앙 기준 폴백
